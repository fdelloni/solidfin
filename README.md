# SolidFin

Landing y prototipo de UI para **SolidFin**, una plataforma de préstamos con garantía solidaria grupal (en desarrollo). Incluye un simulador de préstamo y una vista de app.

**Estado:** prototipo / work in progress — página HTML estática, sin backend.

Abrir `index.html` en el navegador — no requiere build ni dependencias.
